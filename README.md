## :package: Installation
Clone or [Download](https://github.com/Defkil/todo-python-client/archive/main.zip) this project
```sh
git clone https://github.com/Defkil/todo-python-client.git
```
Open a terminal in the project folder and install the NPM dependencies.
```sh
npm install -D
```

## :hammer_and_wrench: Usage
Build production code
```sh
npm run build
```

Build development code on file changes
```sh
npm run watch
```
